Tone

Voice & Tone Guide

We're the friend who always knows the best gigs in town. Sharp, warm, never corporate-boring.

⚡
Energetic, Never Aggressive
Match the energy of a live event. Bring enthusiasm without desperation.
✓ "Only 47 seats left. Move fast."
✗ "BUY NOW BEFORE IT'S TOO LATE!!!"
🤝
Warm, Not Sycophantic
Talk like a knowledgeable friend. Never like a salesperson.
✓ "Great choice! You're going to love this one."
✗ "Dear Valued Customer, thank you for your interest."
🎯
Specific, Not Vague
Give real details. Numbers, times, venue names. Vague copy erodes trust.
✓ "Doors open 6:30 PM. Parking at Gate 3."
✗ "Event starts in the evening. Please arrive accordingly."
😄
Playful, Never Silly
Wit earns attention. Silliness loses credibility. Know the line.
✓ "Your cat can't come. But you should."
✗ "OMG this show is sooo amazinggg!!!1!"
🔒
Trustworthy, Not Pompous
Handle money moments (checkout, refunds) with calm authority and zero drama.
✓ "Refund processed. ₹1,499 back in 5–7 days."
✗ "We sincerely apologize for any inconvenience caused."
🌏
Indian at Heart
Reference Indian culture naturally — don't force it or make it cringe.
✓ "Diwali gigs? We've got your mithai + music sorted."
✗ "India is a land of great cultures and festivals!"

Usage Rules

Do's & Don'ts

Brand integrity is product quality. These rules protect the experience.

Do
  • Use Space Grotesk for all headings and display text
  • Maintain minimum 44px touch target for all interactive elements
  • Use the full wordmark "INDTIX" (not "Ind-Tix", "IndTix", "indTIX")
  • Show GST amounts separately in all financial contexts
  • Use the brand purple (#6C3CF7) for primary actions only
  • Keep button copy outcome-oriented ("Book Now", "Get Tickets")
  • Respect the 8px base spacing grid (8, 16, 24, 32, 48, 64px)
  • Use dark backgrounds (#0A0A14 or #0F0F1E) as primary canvas
  • Credit as "Powered by INDTIX — Oye Imagine Pvt Ltd"
Don't
  • Never stretch or distort the logo horizontally or vertically
  • Never place the logo on busy photographic backgrounds without a container
  • Never use red (#FF0000) — use Neon Coral (#FF4F6E) for urgency
  • Never use ALL CAPS for body text (labels only)
  • Never fake scarcity — only show "Selling Fast" when >70% sold
  • Never display prices without inclusive GST disclosure
  • Never use Comic Sans, Papyrus, or any other non-brand font
  • Never show raw API errors to users ("500 Internal Server Error")
  • Never abbreviate "Oye Imagine" to "OI" in customer-facing copy
